# Node

Estrutura de pastas de projetos em Node devem ser organizadas da seguinte maneira

```
Project
├── src
│   ├── index.js
│   ├── Router.js
│   ├── routes
│   │   └── UserRoutes.js
│   │   └── ...
│   ├── controllers
│   │   └── UserController.js
│   │   └── ...
│   ├── model
│   │   └── User.js
│   │   └── ...
│   ├── services (optional)
│   │   └── UserService.js
│   │   └── ...
│   ├── database
│   │   └── Connection.js
│   │   └── UserSchema.js (optional)
│   │   └── ...
│   ├── config (optional)
│   │   └── ...
│   ├── utils (optional)
│   └── └── ...
└── .gitignore
```
