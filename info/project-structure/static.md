# Projetos estáticos
 
 Estrutura de pastas de projetos em estáticos devem ser organizadas da seguinte maneira
 
 ```
 Project
├── assets
│   ├── icons
│   │   ├── icon.jpg
│   │   └── ...
│   └── images
│   │   ├── image.png
│   │   └── ...
├── css
│   ├── main.css
│   └── ...
├── html
│   ├── page.html
│   └── ...
├── js
│   ├── main.js
│   └── ...
└── index.html
 ```
